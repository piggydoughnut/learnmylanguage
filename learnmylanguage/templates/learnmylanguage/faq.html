1. How do you get data?

Duolingo has an API which serves users data. There is no personal data (e.g. email, real name), only duolingo study related data

2. How can i stop playing?

If you stop using Duolingo we will notice that you broke your streak and will send you one notification. If you will not start studying again the next day, we consider you a quitter. You will receive no more notifications from us.
